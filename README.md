# PostgreTS

A plug-and-play PostgREST ESM module. Use `deno task make` to transpile as JavaScript, or test queries on the command line via `deno task test <pgrst_query>`.

You may also import the module directly from `https://raw.githubusercontent.com/Starscade/PostgreTS/main/esm/postgrets.esm.js`, but this is not recommended.
